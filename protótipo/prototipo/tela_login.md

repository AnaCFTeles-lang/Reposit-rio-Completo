# 📄 Tela de Login

## 🎯 Objetivo
Permitir acesso seguro e diferenciado para Aluno, Professor ou Administrador NAPNE.

## 🧱 Elementos da tela
- **Título principal:** Sistema de Acompanhamento Acadêmico – NAPNE IFRS
- **Campo 1:** Usuário / E‑mail institucional  
  *Ex: maria.silva@aluno.ifrs.edu.br*
- **Campo 2:** Senha  
  🔒 Botão de alternar exibição (mostrar/ocultar)
- **Botão principal:** ✅ Entrar
- **Link auxiliar:** “Esqueci minha senha?”
- **Barra de acessibilidade (canto superior):**
  - A+ → Aumentar fonte
  - A‑ → Diminuir fonte
  - ◐ → Ativar/desativar alto contraste

## ⚙️ Regras e mensagens
- ✅ Dados corretos → redireciona conforme perfil:
  - Aluno → `tela_aluno.md`
  - Professor → `tela_professor.md`
  - Administrador → área de gestão
- ❌ Dados incorretos → mensagem: *“Usuário ou senha inválidos. Verifique e tente novamente.”*
- ⚠️ Campo vazio → aviso: *“Preencha todos os campos obrigatórios.”*

## 📐 Layout
- Fundo claro / alto contraste disponível
- Campos centralizados, espaçados
- Sem menus laterais — foco só no acesso
